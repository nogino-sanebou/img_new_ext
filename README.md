# 画像を可逆圧縮の練習
bmp ファイルを可逆圧縮・解凍する処理の練習です。

# 今できる事
RAWブロックとRLEブロックに分ける。

- RAW: 先頭バイトに書かれた指定の長さ文入力を読み込み出力する
- RLE: 先頭バイトに書かれた値分だけ次の値を複製して出力する

# 問題点
## 2026.06.17
だいぶサイズは小さくなったが、まだファイルサイズが若干大きい。
6000KB(オリジナル) → 6200KB(変換後)くらい

## 2026.06.16
連続する同一バイトが少ない程
かなりデータ容量が増えてしまう。
試しに使ったデータ(img.bmp)だと6000KB → 10,000KBくらいになってしまう。
